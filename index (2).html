🍪 NSE Cookie Setup

NSE requires a valid browser session cookie. Without it, the dashboard uses sample data. The cookie refreshes periodically — update it if data stops loading.
📋 How to get: Open nseindia.com → F12 → Network tab → click any request → Request Headers → copy the entire cookie value
Required for browser. allorigins.win is free. Or self-host a proxy on Render/Vercel.
3–5s for live market hours. 30s+ when market closed.

🚀 GitHub Pages Deploy

✅ This is a single self-contained HTML file — no server needed. Deploy directly to GitHub Pages!
  1. Download this file using the button below
  2. Go to your GitHub repo: maheshmaximusmax/Nifty-Dashboard
  3. Upload / commit the file as index.html to the main branch
  4. Go to repo Settings → Pages → Source: main branch / root folder
  5. Your live URL: https://maheshmaximusmax.github.io/Nifty-Dashboard
  6. Open the live site → Settings tab → paste NSE cookie → Save
  7. Data refreshes every 5 seconds automatically!
