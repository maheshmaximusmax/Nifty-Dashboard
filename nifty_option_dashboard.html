🚀 GitHub Pages Deployment

✅ This dashboard is a single HTML file. Deploy it to GitHub Pages by following the steps below.
  1. Download this HTML file (right-click → Save As)
  2. Create a new GitHub repository (e.g., nifty-dashboard)
  3. Upload the file, rename it to index.html
  4. Go to Settings → Pages → Source: main branch / root
  5. Your dashboard will be live at https://username.github.io/nifty-dashboard
  6. Open it, go to Admin Settings and paste your NSE cookie

📡 Data Source

Data fetched every 5 seconds. NSE requires valid session cookies. The dashboard auto-handles JSON parsing and error recovery.
